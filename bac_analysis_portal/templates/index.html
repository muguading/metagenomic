Batch Import

批量导入样本

支持 xlsx、csv、tsv。建议先下载模板，再按列填写样本信息后统一导入。

1. 选择模板格式

人工填写优先下载 XLSX；程序生成用 CSV；命令行或管道处理用 TSV。

准备

先下载模板，填写样本主档和 final_fasta_path，再回到下一步上传文件。

推荐 XLSX

包含数据页、填写说明和词表示例，最适合手动整理样本主档。

2. 上传填写后的文件

首行为字段名，至少包含 sample_name 和 final_fasta_path。选择文件后会先显示预检报告。

系统主档词表

Reference Batch Import

批量导入参考基因组

支持 xlsx、csv、tsv。建议先下载模板，按列填写后一次性导入多个基因组。

1. 选择模板格式

人工维护参考库优先下载 XLSX；程序批量生成记录用 CSV；命令行清洗后导入用 TSV。

准备

先下载模板，填写物种名称、基因组名称和 FASTA 路径，再上传填写后的文件。

推荐 XLSX

包含字段名、中文说明和示例行，适合人工补全物种、基因组名称和 FASTA 路径。

2. 上传填写后的文件

首行为字段名，至少包含 host_name、genome_name 和 fasta_path。选择文件后会先显示预检报告。
